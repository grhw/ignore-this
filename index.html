What is a vocaloid

A Vocaloid is a type of synthesiser, usually used for music, which uses a robotically created voice based on a real person's recordings.

More Info

Vocaloid Software

The Offical vocaloid software started from just 'Vocaloid' (now Vocaloid6). But there are many other vocaloid-like software which are still considered vocaloids. For example, SynthV and UTAU. These software are usually used for Japanese styled music, but you can produce English music if the voice supports it.
